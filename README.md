#**Simple RESTFUL API with Node and PostGre**

## Global Dependencies
* Node > 6
* pssql
* express-generator

## Want to use it
* Install dependencies
* Start psql  - ```psql-f hobbies.sql```
* Run ```npm install```
* Start the server - ```npm start```
